public function addPhase(phaseName:String, order:String, loopback:String = loopback_after_item):void

Adds a validation phase.

The processing order of phases equals the insertion order.

Parameters

phaseName:String — Unique name of the phase.
 
order:String — Phase order (I10N.TOP_DOWN or I10N.BOTTOM_UP).
 
loopback:String (default = loopback_after_item) — Property indicates when the validation cycle should rewind to the first phase.

registerDisplayObject()method 
public function registerDisplayObject(displayObject:DisplayObject, adapter:I10NAdapter):void

Registers a display object.

You need to start the service before, otherwise an error is thrown.

You cannot register the same object twice. An error is thrown if you try.

You cannot reuse an adapter for another display object. An error is thrown if you try.

Parameters

displayObject:DisplayObject — The display object to register.
 
adapter:I10NAdapter — The adapter defining the communication between display object and I10N.

start()method 
public function start():void

Starts the service.

You need to have added at least one validation phase. Otherwise an error is thrown.

unregisterDisplayObject()method 
public function unregisterDisplayObject(displayObject:DisplayObject):void

Unregisters a display object.

You usually call this method before removing a display object from the application.

Any references to that object are being removed when called.

Parameters

displayObject:DisplayObject — The display object to unregister.
