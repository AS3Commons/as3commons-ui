registerDisplayObject()method 
public function registerDisplayObject(displayObject:DisplayObject, adapter:LifeCycleAdapter):void

Registers a display object.

You cannot register the same object twice. An error is thrown if you try.

You cannot reuse an adapter for another display object. An error is thrown if you try.

Parameters

displayObject:DisplayObject — The display object to register.
 
adapter:LifeCycleAdapter — The adapter defining the communication between display object and LifeCycle.

unregisterDisplayObject()method 
public function unregisterDisplayObject(displayObject:DisplayObject):void

Unregisters a display object.

You usually call this method before removing a display object from the application.

Any references to that object are being removed when called.

Parameters

displayObject:DisplayObject — The display object to unregister.

public function validateNow():void

Validates all invalid objects immediately.
